Arty 1.0.22 — Diagnostic connexion Google

• Fix : si le plugin Google natif se bloque (popup s'ouvre mais ne revient jamais), un timeout de 30s force un message d'erreur au lieu de spinner pour l'éternité.
• UI : les erreurs de connexion Google sont maintenant affichées dans un vrai banner rouge plein-largeur sous le bouton (au lieu d'un petit texte facilement raté).
• Logs : chaque étape du flow login est loggée en console (visible via Chrome DevTools remote — chrome://inspect) pour diagnostiquer les futurs bugs.
